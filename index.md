# How to

Cloning the repo

```
git clone https://github.com/fabiocogno/metasploit-modules.git
```

Copy the content to the local Metasploit directory


```
mkdir -p ~/.msf4 && cp -r metasploit-modules/.msf4/* ~/msf4/
```
